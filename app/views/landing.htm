Inexor

Inexor is a fork of the quake-like FPS Cube 2: Sauerbraten. It adds a lot of functionality and tries
to stay open for improvements and suggestions. The task is to be more flexible and create an
environment where developement is easy and fast and creativity can prosper.

Features

  • Entirely free and open source
  • Great development setting
  • Fast pace gameplay
  • Ingame multiplayer map maker

The current release (Pyrimethamine Alpha Edition) can be obtained from GitHub.
